# Skills Manager

一个用于管理 pa-skills 的 Visual Studio Code 扩展。

## 功能特性

- ✅ 查看已安装的技能（全局/本地）
- ✅ 美观的卡片式 UI 界面
- ✅ 技能详情查看（包含文件树）
- ✅ 安装新技能（支持 Zip 和目录）
- ✅ 卸载技能
- ✅ 搜索和筛选技能
- ✅ 点击文件在编辑器中打开
- ✅ 响应式设计

## 快速开始

1. 在 VSCode 中打开此项目
2. 按 `F5` 启动调试
3. 在新窗口侧边栏找到 "Skills Manager" 图标
4. 开始管理你的技能！

## 使用说明

### 查看技能
- 打开侧边栏的 "Skills Manager" 面板
- 所有已安装的技能会以卡片形式展示
- 卡片显示技能名称、描述、版本等信息

### 技能详情
- 点击任意技能卡片查看详情
- 详情页包含：
  - 技能概述
  - 元数据信息
  - 文件树结构（可点击打开文件）

### 安装技能
1. 点击顶部 "➕ 安装技能" 按钮
2. 选择 Zip 包或技能目录
3. 等待安装完成

### 卸载技能
1. 进入技能详情页
2. 点击 "卸载技能" 按钮
3. 确认卸载操作

### 搜索筛选
- 使用顶部搜索框按名称搜索
- 使用下拉菜单筛选全局/本地技能

## 技术栈

- Node.js
- VSCode Extension API
- pa-skills CLI

```

├── extension.js           # 扩展主入口
├── services/

├── .vscode/
│   └── launch.json        # 调试配置
└── README.md              # 本文件
```

## 依赖项

- pa-skills CLI 工具（需要在系统 PATH 或项目目录中）
- Node.js >= 18

## 开发

### 调试
1. 按 `F5` 启动扩展开发主机
2. 在新窗口中测试功能
3. 修改代码后按 `Ctrl+R` 重新加载窗口

### 日志
- 打开 "帮助 > 切换开发人员工具"
- 查看控制台输出
