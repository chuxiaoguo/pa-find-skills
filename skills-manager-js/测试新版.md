## 🎯 如果找不到

### 方法 1：使用命令面板
1. 按 `Ctrl+Shift+P`
2. 输入：`Skills Manager`
3. 选择：**`Skills Manager: 打开面板`**
4. 回车

### 方法 2：切换到资源管理器
1. 按 `Ctrl+Shift+E`（切换到资源管理器）
2. 查看左侧面板
3. 向下滚动到底部

## 💡 这次修改的关键

**之前（自定义图标）**：
```json
"viewsContainers": {
  "activitybar": [
    { "id": "skillsManagerContainer", ... }
  ]
},
"views": {
  "skillsManagerContainer": [...]
}
```
❌ 可能因为主题/设置问题看不到图标

**现在（使用资源管理器）**：
```json
"views": {
  "explorer": [
    { "id": "skillsManagerView", ... }
  ]
}
```
✅ 直接在资源管理器底部，100% 可见
