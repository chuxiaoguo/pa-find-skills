# 📍 图标位置详细说明

## ⚠️ 重要：Activity Bar 不是侧边栏！

很多用户混淆了 Activity Bar 和侧边栏（Sidebar），请看下面的说明：

```
┌─────────────────────────────────────────────────────────┐
│  VSCode 窗口布局                                        │
├──┬───────────────────────────────────────────────────┤
│活│ 侧边栏 (Sidebar)                                    │
│动│  ┌───────────────┐                                 │
│栏│  │ 📁 文件管理器 │  ← 这是侧边栏内容              │
│A │  │ 🔍 搜索      │                                 │
│c │  │ ⚡ 源代码管理 │                                 │
│t │  │ 🐛 调试      │                                 │
│i │  │ 📦 Skills    │  ← 点击后这里显示内容          │
│v │  └───────────────┘                                 │
│i │                                                     │
│t │                                                     │
│y │                                                     │
│y │                                                     │
│  │                                                     │
├──┴───────────────────────────────────────────────────┤
│  主编辑器区域                                          │
│  （你之前看到诊断页面的地方）                           │
└─────────────────────────────────────────────────────┘
```

## 🎯 如何找到图标

### 方法 1：使用命令面板（最简单）✅

1. 按 **`Ctrl+Shift+P`** 打开命令面板
2. 输入：**`Skills Manager`**
3. 选择：**`Skills Manager: 显示侧边栏`**
4. 回车

### 方法 2：在 Activity Bar 中查找

1. **看向 VSCode 窗口的**最左侧**边缘**
2. 应该能看到一条垂直的图标栏
3. **向下滚动**这条栏（鼠标滚轮）
4. 查找 **📦 盒子图标**
5. 点击这个图标

### 方法 3：切换侧边栏

有时候侧边栏被隐藏了：

1. 按 **`Ctrl+B`** 多次
2. 侧边栏会在隐藏/显示之间切换
3. 同时观察最左侧的 Activity Bar

### 方法 4：使用开发者工具

1. 按 **`Ctrl+Shift+I`** 打开开发者工具
2. 切换到 **Console** 标签
3. 输入以下命令查看所有活动栏图标：
   ```javascript
   // 查看所有视图容器
   ```
4. 查看输出

## 🔍 调试命令

我已经添加了调试命令，你可以：

1. 按 **`Ctrl+Shift+P`**
2. 输入：**`Skills Manager: 调试命令`**（如果有的话）
3. 或者直接输入：**`Skills Manager`**
4. 选择相关命令
