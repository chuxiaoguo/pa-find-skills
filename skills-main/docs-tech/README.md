### 🔧 技术栈

- **运行时**: Node.js >= 18
- **语言**: TypeScript 5.9
- **包管理**: pnpm 10.17.1
- **构建工具**: obuild 0.4.22
- **测试框架**: vitest 4.0.17
- **交互式提示**: @clack/prompts 0.11.0
- **Git 操作**: simple-git 3.27.0
- **YAML 解析**: gray-matter 4.0.3

```

  │   ├── skills.ts (技能发现)
  │   ├── installer.ts (安装引擎)
  │   ├── agents.ts (代理管理)

  ├── remove.ts (移除技能)
  │   ├── installer.ts
```

### 🎯 设计原则

1. **跨代理兼容**: 支持 42+ 种不同的 AI 编码代理
2. **单一真相源**: 使用符号链接避免重复
3. **向后兼容**: 保持稳定的公共 API
4. **安全优先**: 防止路径遍历攻击
5. **用户体验**: 交互式提示和清晰的错误消息
