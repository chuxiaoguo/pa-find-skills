# React Components Generator

基于自然语言描述生成 React 组件代码。支持 TypeScript、Hooks、Styled Components 等现代 React 开发技术栈。

## 功能特性

- 自然语言转 React 组件
- 支持 TypeScript
- 支持 Hooks
- 支持多种样式方案

## 使用方法

1. 描述你需要的组件
2. 选择技术栈
3. 生成组件代码
