### 🔧 技术栈

- **运行时**: Node.js >= 18
- **语言**: TypeScript 5.9
- **包管理**: pnpm 10.17.1
- **构建工具**: obuild 0.4.22
- **测试框架**: vitest 4.0.17
- **交互式提示**: @clack/prompts 0.11.0
- **Zip 处理**: extract-zip 2.0.0
- **YAML 解析**: gray-matter 4.0.3
- **HTTP 请求**: native fetch API

### 🎯 设计原则

1. **内网优先**: 适配公司内网环境，无法访问外部服务
2. **最小改动**: 复用 skills-main 优秀代码，只改动必要部分
3. **渐进式迁移**: Mock 接口先行，数据格式保持一致
4. **向后兼容**: 保留本地文件和 Zip 安装方式
5. **安全可靠**: Token 加密存储，防止路径遍历攻击

### 🔄 与原版的主要差异

| 模块 | skills-main | pingancoder-skills | 变更程度 |
|------|-------------|-------------------|----------|
| 技能来源 | skills.sh API | 公司内网静态资源服务 | 🔄 完全重写 |
| 代理数量 | 42+ | 4 (gemini/opencode/openclaw/pingancoder) | ✂️ 大幅精简 |
| 提供者数量 | 7+ | 3 (API/本地路径/本地Zip) | ✂️ 精简 |
| 下载方式 | Git clone | API 下载 + 本地文件 | 🔄 改造 |
| 认证机制 | 无 | Token 认证 + 本地缓存 | ➕ 新增 |
| 更新检测 | GitHub API | 内网 API | 🔄 改造 |
